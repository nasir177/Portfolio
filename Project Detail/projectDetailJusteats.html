Disrupting the Aggregator Monopoly

In the Indian market, local pizza owners lose significant margins to Zomato and Swiggy. This design strategy eliminates the middleman, allowing owners to use their own local staff for delivery.

By removing a 25-30% commission, the brand can directly reduce pizza prices for the user while maintaining higher profit, creating a sustainable hyper-local logistics model.

Visualizing the D2C Journey

01 // Splash to Immersive Home

Built a high-performance Home screen utilizing advanced Figma prototyping. I implemented a "Pizza Zoom" effect where clicking a category provides a seamless cinematic transition into the item details, reducing perceived load times.

Technical Edge: Smooth Scroll & Latency Optimization

02 // The Direct Menu

Designed for speed and clarity. The menu uses lazy-loading image rituals to ensure that heavy visual assets don't compromise the smooth scroll experience, keeping the user in the "culinary flow" without technical friction.

03 // Trust Architecture (About Us)

Unlike aggregators, D2C requires deep trust. The About Us and Contact pages humanize the local pizza shop, showcasing the chef's expertise and the delivery team's local identity to build long-term loyalty.
