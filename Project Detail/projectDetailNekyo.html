01. Design Philosophy

The Problem

Japanese is intimidating for beginners. Dry academic interfaces and lack of native pronunciation feedback create a high barrier to entry, causing learners to stall before reaching proficiency.

The Solution

A multi-sensory platform that combines Audio API integration, Stroke Practice Boards, and JSON-driven Conversations to provide a seamless learning journey from basic scripts to real-world chat.

02. Script Mastery Page

Hiragana, Katakana & Kanji

The foundation of the app. clicking any character triggers instant native pronunciation via an integrated Audio System. Users can also visualize and practice the precise stroke order for each script directly on the responsive canvas, ensuring correct calligraphy from day one.

Feature: Instant Audio & Stroke Validation

03. The Learning Hub

9 Structured Vocabulary Chapters

A comprehensive curriculum divided into 9 chapters. Each chapter uses interactive quizzes to reinforce vocabulary retention. This data-driven approach allows students to track progress as they move from basic greetings to complex nouns.

04. Grammar Video Library

Curated Expert Instruction

Recognizing that grammar requires deep explanation, I architected a dedicated Video Page. It sources high-quality lessons from the best Japanese educators on the web, centralizing the most reliable resources into one distraction-free classroom.

05. Real-World Interaction

Bot Chat & Custom Keyboard

I created a custom On-Screen Japanese Keyboard to facilitate real-time chat with a JSON-driven bot. This allows students to practice phrasing and slang without installing complex software, providing a safe feedback loop for conversational skills.

06. Digital Practice Board

Virtual Green Board Classroom

Modeled after a traditional classroom environment, this module allows for free-hand practice. Using mobile touch-events or PC mouse-tracking, students can draw what they have learned, bridging the gap between digital recognition and physical writing skills.

07. Health & Accessibility

Language acquisition requires sustained screen time. I implemented a robust Light/Dark Mode toggle designed with student ocular health in mind, managing contrast ratios for late-night study rituals without causing eye fatigue.

Mode Comparison

Visual Harmony across both Spectrums

08. Final Retrospective

BCA Engineering Edge

Leveraging my technical background, I optimized the Audio API and canvas practice modules for low-latency performance. Switching to TypeScript ensured that the character and quiz data models remained strictly typed and scalable for future Kanji expansion.

Product Result

Nekyo provides a seamless design to production solution that removes dry academic barriers. By focusing on writing, listening, and real-time interaction, the app delivers a multi-sensory ritual for serious language beginners.
