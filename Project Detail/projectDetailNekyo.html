Phase 01: Research

User Problem & Research

Japanese learning is often gated by intimidating 90s-style interfaces. As a JLPT N4 proficiency student, I identified that learners stall because current tools offer dry static tables without sensory feedback.

The goal was to solve ocular fatigue and boost retention by bridging the gap between visual character recognition, auditory memory, and physical stroke practice.

Phase 02: Design & Engineering Solution

01 // Audio Script Mastery

Designed a unified module for Hiragana, Katakana, and Kanji. Native pronunciation is triggered on-click using custom Audio API integration, ensuring correct phonetics from the start.

UI/UX + Frontend Implementation

02 // Chat Bot & Custom Keyboard

Built a JSON-driven bot conversation page paired with an on-screen Japanese keyboard. This removes the friction of installing language packs while allowing users to practice real-time sentence construction.

Logic Flow Design

03 // Structured Learning Hub

Divided the curriculum into 9 progressive chapters. Implemented data-driven interactive quizzes that validate retention before the user advances to the next stage.

Information Architecture

04 // Virtual Practice Board

Designed a "Virtual Green Board" using HTML5 Canvas. It allows for free-hand stroke practice via mobile touch or mouse, mimicking the muscle memory of a physical classroom.

Gesture-Based UX

Retrospective & Logic

Coding Implementation

Developed with React and TypeScript to ensure scalable data models. Optimized the Audio API and canvas logic for ultra-low latency interaction on mobile devices.

Design Impact

By replacing standard learning hurdles with sensory feedback, Nekyo created a ritual-based learning experience that reduces beginner bounce rates.
