01. Overview

Nekyo is a Japanese learning web application built with React and then carefully migrated to TypeScript to follow modern best practices for type-safe frontend development. It targets anime fans, students, and selfâ€‘learners who want a friendly way to practice Hiragana, Katakana, Kanji, and basic conversations.

02. Problem

Most beginner Japanese learners struggle with motivation and consistency because many apps feel either too academic or poorly structured. On the development side, JavaScript-only codebases in learning apps can become hard to maintain as features grow, especially when handling multiple content types like characters, vocabulary, and quizzes. Nekyo solves both issues by combining a playful, Japanese-inspired UI with a strongly-typed TypeScript codebase that makes the app reliable and easy to extend.

03. Design Process

The UX process started by mapping core learning journeys: land on the hero section, pick a script (Hiragana, Katakana, Kanji), practice characters, and then test understanding through vocabulary and conversation quizzes. The Home/Hero section uses a full-screen video background with a mascot and primary CTA, making the app feel like an inviting, anime-style experience rather than a dry textbook. From there, each major learning mode was broken into focused UIs: a character practice screen for step-by-step recognition, a practice board for repetitive writing/reading drills, and a conversation quiz screen for applying phrases in context. Navigation, hierarchy, and spacing were tuned so users always know what to do next, with clear buttons like "Start," "Next," and "Check answer" and consistent layouts across devices.

Pictured above: Nekyo learning interface showcasing the Home screen with character practice modules, Learn section with structured lessons and quizzes, Practice board for handwriting drills, Videos section for grammar resources, and the Conversation screen with custom kana keyboard for interactive dialogue practice.

04. Key Screens & Interactions

  • Home/Hero section: Full-screen video hero with Japanese visuals, app title, and CTA leading into learning modules. This screen establishes brand identity and emotional connection from the first second.
  • Character practice UI: Separate views for Hiragana, Katakana, and Kanji where each character is shown with reading and meaning. When users click on any character, an audio pronunciation plays automatically, helping them learn the correct sound for each kana and kanji. Typed data models drive the character lists, making it easy to iterate over characters and ensure consistent rendering with integrated audio playback.
  • Learn: The Learn area is structured into clear lessons and quizzes so users can build knowledge step by step. Each lesson introduces focused topicsâ€”such as Hiragana, Katakana, basic Kanji, and core vocabularyâ€”and then immediately reinforces them with short quizzes to check recognition and understanding. This keeps the experience active rather than passive, helping learners move from "seeing a character" to actually recalling it when asked.

    Quizzes are integrated directly into the lesson flow, so learners always know what to do next: study, attempt questions, review feedback, and then continue to the next topic. This creates a smooth, course-like progression that feels more like a guided path than a random collection of screens.
  • Practice board: An on-screen canvas where learners can actually draw each character with their finger or mouse, just like writing in a Board. They can clear the canvas to try again or export their practice as PNG/JPG and save it on their device for later review.
  • Videos: The Videos section curates high-quality external video content focused on Japanese grammar and foundational language concepts. Instead of asking learners to search the internet themselves, Nekyo brings the best grammar explanations into one place, saving time and reducing confusion about where to start.

    These videos complement the app's interactive practice: users can watch a grammar explanation (for example, sentence structure or particles), then switch back to characters, vocabulary, or conversation practice to apply what they just learned. This combination of structured lessons, quizzes, and expert video content makes Nekyo feel like a compact, well-organized learning hub rather than just a simple character drill app.
  • Conversation screen: A guided chat-style interface where learners answer practical questions like "How do you say 'Good morning' in Japanese?" using a custom on-screen kana keyboard. The layout clearly separates the chat bubble, answer input, and keyboard, so users can focus on building sentences while the typed data models keep each prompt, answer, and validation step consistent and easy to maintain.

05. Tech Stack & Implementation

Nekyo runs entirely on the frontend using:

  • TypeScript (primary language) for type safety and maintainable code.
  • React for component-based UI and state management.
  • Tailwind CSS Modules for scoped, maintainable styling.
  • Local JSON files as the content source for characters, vocabulary, and quizzes, keeping the app lightweight and serverless.
  • Vercel for fast, production-like hosting.
